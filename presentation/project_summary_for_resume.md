# A/B-тест карточки объявления: SQL, EDA и статистическая проверка
Учебный продуктовый аналитический проект на синтетических данных.

- Собрала SQLite-базу с таблицами `users`, `ads`, `events`, `payments` и написала SQL-запросы для расчёта воронки, продуктовых метрик и user-level витрины.
- Провела EDA, проверила баланс групп, подготовила датасет на 12 000 пользователей для статистической проверки.
- Реализовала z-test для contact conversion, bootstrap для ARPU и Mann-Whitney U test для revenue.
- Получила статистически значимый рост contact conversion: 28.60% → 30.60%, uplift 2.00 п.п., p-value = 0.0164.
- Сформулировала продуктовую рекомендацию: rollout возможен, но требует проверки качества контактов и бизнес-рисков.

Стек: Python, pandas, NumPy, SciPy, SQLite, SQL, Jupyter Notebook, EDA, A/B-тестирование.
