# Slide 2. Бизнес-вопрос

В сервисе объявлений изменилась карточка объявления и кнопка контакта с продавцом.

Нужно понять, стоит ли выкатывать тестовую версию на всех пользователей.

Главная метрика: доля пользователей, которые связались с продавцом.

Дополнительно смотрим выручку, покупки и платящих пользователей.

---

# Slide 3. Данные и БД

Данные синтетические, проект учебный.

В проекте четыре основные таблицы:

- `users`;
- `ads`;
- `events`;
- `payments`.

Создана SQLite-база. Через SQL рассчитаны размеры групп, событийная воронка, основные метрики и user-level датасет.

Итоговый датасет для статистики: 12 000 пользователей.

---

# Slide 4. Проверка эксперимента и EDA

Группы сбалансированы: 6 000 пользователей в `control` и 6 000 в `test`.

Проверены распределения по региону, устройству и сегменту.

Явного перекоса между группами не видно.

Выручка разреженная и скошенная: большинство пользователей ничего не платит. Поэтому по ней нельзя делать вывод только по среднему.

---

# Slide 5. Основные метрики

Основная метрика `has_contact`:

- control: 28.60%;
- test: 30.60%;
- абсолютный uplift: 2.00 п.п.;
- относительный uplift: 6.99%.

ARPU:

- control: 14.01;
- test: 15.09.

В тестовой группе контактная конверсия выше. По выручке тоже есть небольшой рост, но его нужно проверять статистически.

---

# Slide 6. Статистическая проверка

Методы:

- для contact conversion использован z-test для двух долей;
- для ARPU использован bootstrap разницы средних;
- для revenue дополнительно использован Mann-Whitney U test.

Результаты:

- p-value для `has_contact`: 0.016408;
- 95% bootstrap CI для ARPU: [-2.18; 4.40];
- bootstrap p-value для ARPU: 0.527800;
- Mann-Whitney p-value для revenue: 0.765537.

Рост contact conversion статистически значим. По ARPU статистически значимого эффекта нет.

# Slide 7. Рекомендация

Тестовую версию можно рекомендовать к rollout, потому что она статистически значимо повышает контактную конверсию.

При этом по выручке значимого эффекта не найдено.

Перед реальным rollout нужно проверить:

- корректность эксперимента;
- стабильность эффекта по сегментам;
- качество контактов;
- жалобы и негативные эффекты;
- долгосрочное влияние на удержание и выручку.
